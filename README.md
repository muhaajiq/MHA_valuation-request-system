# MHA_valuation-request-system

#Overview
Mini valuation request system using .NET 8 Web API and React 18 with TypeScript.

#Run Backend
1. cd backend
2. dotnet run

#Run Frontend
1. cd frontend
2. npm install
3. npm start

#Assumptions
1. In-memory DB used for simplicity
2. Focus on architecture over UI
