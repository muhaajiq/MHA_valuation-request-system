# MHA_valuation-request-system

#Overview
Mini valuation request system using .NET 8 Web API and React 18 with TypeScript.

#Run Backend
cd backend
dotnet run

#Run Frontend
cd frontend
npm install
npm start

#Assumptions
In-memory DB used for simplicity
Focus on architecture over UI
